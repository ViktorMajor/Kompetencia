# Kompetencia

Ez a projekt egy statikus, HTML és CSS alapú weboldal, amely pszichoedukációs segédanyagokat gyűjt a szülők számára. A honlap négy fő blokkból áll (Óvodáskor, Látencia, Kamaszkor, Válás), valamint gyakori kérdések szekciót tartalmaz.

## Fájlstruktúra
- `index.html` – a fő oldal tartalma.
- `css/style.css` – stíluslap.
- `img/` – ide kerülhetnek képek (a könyvtár jelenleg üres).
- `assets/meselistak.txt` – korcsoportonkénti meselisták.
- `assets/tanacsok.json` – strukturált tanácsok.

A weboldal publikálható például GitHub Pages szolgáltatással.
